MOBILE

Dog Watch

Dog Watch was a 'ONE TAP' app to report sightings of Stray Dogs in your neighborhood and volunteer to be a caregiver. It used GPS to track your location and SMS to send the report. (Freelance)

Mathblitz

Mathblitz aimed at making mental math fun. The App is based on an Adaptive Algo that increases the difficulty level of Mental Math as the game progresses. Equipped with Gamification and Ad Revenue Integration

Trader Geek

Trader Geek was developed to mimick the Entry Level Interview Tests for Junior Trader positions at established Trading Firms. It recieved a humongous response at IIT Bombay.

Joy

Joy was an iOS app developed to act as a bridge between NGO's and donors. The app facilitated a smooth donation process for users wherein they could also choose and monitor where their donations go.

BitAlert

BitAlert was an iOS app developed for Bitcoin Price Trend Analysis and Prediction. We also made a sentimeter by making use of the twitter API as a part of this app.

WEB

Cluster - One tap app for networking events

Cluster is a one tap networking app for networking with individuals at networking events. It helps you exchange business information in one tap and also keeps a record of the context of your meetings with new people!

FootPong - Football + Pong with AI

FootPong was a Chrome App written entirely in Javascript. The game included an AI Computer opponent. Gameplay video here

Comic Book Plugin

ComicJS was a JS and jQuery based User Interface Plugin Application primarily geared towards being used as a Comic Book Display. The Plugin was made such that it could be easily embedded anywhere.Live demo can be found here

Storydock App

Interactive Story Telling App based in JS and jQuery. The idea was to present a 9 point story in an interactive format. The overall goal was to build this app into a community of Authors and Readers. Alpha stage interface was Dock effect + Storyboard (Web). Further development (on Android) is on.

CONSOLE (C++)

Graph Visualizer

GraphViz was a console based program written in C++ to visualize graph structures. It used the Force Layout Algo to properly visualize the graphs. Video can be found here .

Logic Simulator

Built an Event based Logic Simulator based on the Naive Twin Queue Algorithm. This kind of simulator would find applications in the realm of Circuit Simulation, and this project was an Academic One. Was written in C++. Github code is OSS

Evil Hangman

Evil Hangman was a transformed version of the regular Hangman game that had a cheat Algorithm behind the scenes.The Algorithm used a 'Search and Swap' Approach. The game was console based and written in C++.The video can be found here

Random Writer

Random writer was a program that output paragraphs of english text that made more and more sense as the order of the model used was increased. The program used Markov Models and was written in c++.The code and documentation can be found here
